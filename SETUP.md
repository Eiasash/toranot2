# Getting Started

## Prerequisites

- Node.js (v18+)
- npm

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173/Toranot/`).

## Build for production

```bash
npm run build
```

Output goes to the `dist/` directory.

## Run tests

```bash
npm run test
```

## Type check

```bash
npm run typecheck
```
